Data Name: TripAdvisor v1

Data Descriptions:
This data was scripted from online reviews on tripadvisor.com. There is only one context: trip type (Family, Couples, Business, Solo travel, Friends). Other features about users and hotels are available. The data set is pretty sparse in ratings and contexts: 4669 ratings, 1202 users, 1890 hotels.
